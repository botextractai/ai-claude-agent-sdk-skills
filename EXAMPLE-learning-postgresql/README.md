# PostgreSQL Learning Path

Welcome to your comprehensive guide to learning PostgreSQL, the world's most advanced open-source relational database.

## About This Guide

This learning path is designed to take you from complete beginner to confident PostgreSQL user through a structured, progressive approach. Whether you're a developer, data analyst, or system administrator, this guide provides the foundation you need to work effectively with PostgreSQL.

## What You'll Learn

- **Why PostgreSQL?** Understanding the motivation and problems it solves
- **Installation & Setup** Getting PostgreSQL running on your system
- **Core Concepts** Tables, queries, transactions, and ACID compliance
- **Practical Patterns** Real-world database design and optimization techniques
- **Next Steps** Advanced features and ecosystem tools

## How to Use This Guide

1. **Start with `learning-path.md`** - This is your main curriculum, organized into 5 progressive levels
2. **Follow the levels in order** - Each level builds on previous concepts
3. **Practice with code examples** - The `code-examples/` directory contains runnable SQL scripts for hands-on learning
4. **Reference `resources.md`** - Links to official documentation, tutorials, and community resources

## Prerequisites

- Basic command-line familiarity
- No prior database experience required
- Access to a computer (Windows, Linux, or macOS)

## Time Commitment

- **Level 1 (Overview):** 15-20 minutes
- **Level 2 (Installation & Hello World):** 30-45 minutes
- **Level 3 (Core Concepts):** 2-3 hours
- **Level 4 (Practical Patterns):** 2-3 hours
- **Level 5 (Next Steps):** 15-20 minutes

**Total:** Approximately 6-8 hours to complete the entire learning path, plus additional time for practice and exploration.

## PostgreSQL Version

This guide is compatible with PostgreSQL 15 and later. The latest stable version is PostgreSQL 17, but all concepts apply to recent versions. We recommend starting with PostgreSQL 16 or 17 for the best ecosystem support.
