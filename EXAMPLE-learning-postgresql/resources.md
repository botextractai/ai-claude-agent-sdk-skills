### Architecture Overview
The PostgreSQL codebase is organized into modular components:
- **backend/** - Core database engine (parser, optimizer, executor)
- **bin/** - Command-line utilities (psql, pg_dump, etc.)
- **interfaces/** - Client connection interfaces
- **pl/** - Procedural language support
- **tutorial/** - Educational SQL and C examples

- **PostGIS** - Geospatial database extension

- **ParadeDB** - PostgreSQL extension for full-text search (Elasticsearch replacement)
- **FerretDB** - MongoDB API compatibility layer for PostgreSQL
- **TimescaleDB** - Time-series database built on PostgreSQL
